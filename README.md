# DAT 494 - Deep Learning Structures

**Course Description & Topics:**  

Deep learning with artificial neural nets underpins all modern AI, such as large language models and image and video generators. 

This course presents multiple advanced topics in some depth, including:
- Advanced convolutional nets  
- Generative AI  
- Reinforcement learning (RL)  

**Generative models discussed include:**  
- Transformer-based autoregressive language models  
- Generative adversarial networks (GANs)  
- Variational autoencoders (VAEs)  
- Diffusion models  

**Reinforcement learning topics include:**  
- Gymnasium (OpenAI Gym)  
- Temporal difference learning  
- Deep Q learning  
- RL libraries  
- World models  

Emphasis is on building models in Python using PyTorch. The course also covers:
- Implementing workflows using existing high-level libraries  
- Transfer learning with pre-trained models  
- AI agents  
- Retrieval augmented generation (RAG)
